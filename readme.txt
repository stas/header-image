Plugin adds a new section in Media settings panel for handling header images

== Description ==

Plugin adds a new section in Media settings panel,
so you can select any of the uploaded pics and use it as a background image 
for a CSS element (ex. header).

This was designed to fix lack of a header image option for older themes.
